# obsidian--code-fold

obsidian 原生预览视图下无法折叠代码块。

Collapsible Code Block挺好但是它在折叠callout块里面的代码时会把下面的内容一起折叠
其它插件内容太多了不需要，暂时没找到合适的插件。所以写了这个插件，功能简单单一纯粹是为了折叠代码块，并且不影响编辑体验

## 使用方法
启动插件后，预览视图下代码块左上角会出现字符 * 或者 #，点击它们就可以折叠或者展开。

## 安装
下载manifest.json main.js style.css到codeblock-fold文件夹，将此文件夹move到.obsidian/plugin/目录下，在obsidian里面刷新插件，开启即可。

## 其它
如果你想丰富它的功能可以自己把代码下下来去修改成你满意的效果。
